A modern Progressive Web App for managing badminton academy operations.

## Features

- **Student Management** - Add, edit, and track student profiles
- **Attendance Tracking** - Mark daily attendance with batch filtering
- **Performance Analytics** - Comprehensive skill assessments and progress tracking
- **Offline Support** - Works without internet using IndexedDB
- **Mobile First** - Responsive design optimized for all devices

## Tech Stack

- Next.js 16 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- Dexie.js (IndexedDB)
- PWA enabled

## Getting Started

```bash
npm install
npm run dev
```
